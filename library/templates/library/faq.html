Frequently asked questions

Do I need to create an account to be able to read books?
No, this is an open-source, digital library that allows users to read and download books without creating an account.
How can I upload files?
You need to create an account with us so that we can review the file and approve it before making it available for our users.
What file types are allowed?
Only PDFs are allowed at the moment, but we plan on accepting ePUBs in the future!
